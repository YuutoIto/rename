rename [directory] [target-str] [options] replase-option [option's arguments]

`directory` that has the rename target files.
`target` is replase target. you can use regexp ^,$,?,*

replase-options types
targetに一致した文字列を置き換える

置き換え対象の文字列指定に
ABCdef.txt のABC*.txtとすると*にあたる部分が置き換え対象になる

-n [NUM] 連続する数字に置き換える
	[NUM] is first number, and number of digits.
	The default is 0.
	Format: case of 005, first number is 5,  digits is 3
			case of 050, first number is 50, digits is 3
			case of 0,   first number is 0,  digits is auto, no fills character
			
-r STR replase the target to <str>.
	
-e erase the target
	if specified ^ to target, erase from front
	case of $, erase from behind


-? から -?までを-付きでパースする
正規表現は複数同時利用できる
スペースの繰り返しはエスケープが必要?

実装する処理
	変更後の名前のファイルがすでに存在する場合上書きか、_をつけるか、変更しないでそのままか聞く
	変更前と変更後のファイル名をstdoで表示する
	一つずつ変更する文字列を表示して許可をとる

後々の実装予定オプション
	リネーム前にバックアップを取る
	名前の重複時の処理を指定する
	リネーム対象から拡張子を外す
		フィルター処理とバインド処理が必要になる
	ディレクトリ名を変更するモード
	全角対応
